# AMW: we are moving to a 'virtualization' rather than 'deletion' model here
# because we would then have to redefine the mainchain atoms, and if we do that
# the torsional potential wouldn't work for these residues. That MAY be appropriate,
# but it seems like an unnecessary check for now.

NOT VARIANT_TYPE VIRTUAL_PHOSPHATE
